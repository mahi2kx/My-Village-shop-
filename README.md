# 🛒 Laxmi Kiranam Smart Shop

## 🚀 Live Demo

👉 https://laxmi-kiranam-shop.web.app

---

## 💡 Project Overview

This is a **real-world web application** built for a village shop.

Customers can directly check:

* Available products
* Prices
* Stock status

Then visit the shop and purchase easily.

👉 This system is **actively used in a real village environment**, making it a practical solution.

---

## 🔥 Key Features

✔ View products (Wines 🍺, Cool Drinks 🥤, Chicken 🍗)
✔ Add to cart functionality
✔ Admin can update price & stock
✔ Real-time data using Firebase
✔ Mobile responsive design
✔ Simple and easy UI for rural users

---

## 🧠 Tech Stack

* HTML
* CSS
* JavaScript
* Firebase Firestore
* Firebase Hosting

---

## 🌍 Real-World Impact

This project solves a real problem in villages where:

* People don’t know product availability before visiting
* Time and effort are wasted

👉 With this app:
Users can check everything online and directly visit the shop.

---

## 📱 How to Use

1. Open the website
2. Browse products
3. Check price & stock
4. Visit shop and purchase
